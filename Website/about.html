About

This website was designed to dynamically generate pages for the participating Athletes, their respective Countries, and the Events that took place during the 2014 Winter Olympics in Sochi, Russia. Hosted on PythonAnywhere and dynamically generated using Django, the website features a responsive design that is optimized for all viewing resolutions.
